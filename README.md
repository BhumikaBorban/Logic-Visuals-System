AlgoVisual is an interactive web-based suite designed to help students and developers visualize complex data structures and algorithms in real-time.
By combining sorting, searching, and tree structures into a single unified dashboard, this project provides a "thought partner" experience for learning computer science fundamentals.

3. Binary Search
Step-by-Step Logic: Follow the Low, Mid, and High pointers as they narrow down the search space.
Status Logs: Detailed messages explain why the search moves left or right based on the middle element.
